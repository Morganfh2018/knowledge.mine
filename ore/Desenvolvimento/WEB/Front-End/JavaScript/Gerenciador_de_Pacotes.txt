h1. Gerenciador de Pacotes

{info:title=Sobre}Bower é um gerenciador de pacotes para web. Ele oferece uma solução genérica para o problema de gerenciar os pacores de front-end. O Bower utiliza e executa sobre Git e é agnóstico de pacote.{info}

{note:title=Dependência}Precisa e executa sobre o [cpi:Node.js]{note}

* [Bower|http://bower.io/] \- A package manager for the web, by Twitter

* \[GitHub\] [bower / bower|https://github.com/bower/bower]

* [Bower components|http://sindresorhus.com/bower-components/] \- Sistema de busca dos pacotes/componentes indexados pelo Bower

* \[YouTube\] [Conheça o Bower|https://www.youtube.com/watch?v=0XffYRF7-G0]
