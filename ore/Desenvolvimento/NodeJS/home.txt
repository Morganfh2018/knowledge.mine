h1. Node.js

{tip:title=Motivação}Um dos motivos do interesse pelo Node.js é a possibilidade de executar códigos JavaScript sem a necessidade de um Web Browser. Com isto e sobre o Node.js surgiram muitas ferramentas de [automatização|cpi:JS - Executor de Tarefas], qualidade e [teste|cpi:JS - Testes] de software para projetos FrontEnd.

\\

Um exemplo de uso do Node.js com o Jenkins CI pode ser encontrado no : [ci.angularjs.org|http://ci.angularjs.org/] | [configurações|https://github.com/angular/ci.angularjs.org] - o qual é utilizado pelo framework MVC JavaScript [cpi:Angular.js]
